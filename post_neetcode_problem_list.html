 Hard Most commonly asked "hard" at FAANG new grad. Two-pointer O(1) space solution is the expected answer, not the stack version. Move the smaller side inward. If the current height isn't a new max, the difference between it and the running max is trapped water.

Backtracking / grid DFS
 Med Grid DFS with in-place visited marking. Very commonly asked, easy to mess up the backtracking undo step. DFS from every cell that matches word[0]. Expand to neighbors matching the next character, marking cells '#' to block revisits. Restore on backtrack.
